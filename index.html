Ayuda rápida — Contar elementos y expresiones

Puedes contar elementos de una lista usando len(nombre_lista). Las expresiones se evalúan en condiciones, asignaciones y plantillas {{ ... }}.

Operadores soportados

  • Aritméticos: +, -, *, /
  • Comparación: ==, !=, >, <, >=, <=
  • Lógicos: &&, ||, !
  • Ternario: cond ? a : b
len(items) > 0 && startsWith(name, 'A') ? 'OK' : 'KO'

Funciones disponibles (con ejemplos)

  • len(x): longitud de string o lista — len(items)
  • split(text, 'sep'): string→lista — split('a,b,c', ',')
  • join(list, 'sep'): lista→string — join(items, ';')
  • toNumber(x): a número — toNumber('42')
  • parseInt(x), parseFloat(x) — parseInt('10')
  • bool(x): coerciona a boolean — bool('yes')
  • trim(s): recorta espacios — trim(' hola ')
  • contains(hay, needle) — contains(name, 'ana')
  • startsWith(s, pref), endsWith(s, suf) — startsWith(code,'ES-')
  • upper(x), lower(x) — upper(name)
  • isEmpty(x): vacío/null — isEmpty(email)
  • coalesce(a,b,c,...): primer valor no vacío — coalesce(alias, name, 'N/A')
Composición:
len(split('a,b,c', ','))   // 3
join(split('x|y|z','|'), ', ') // "x, y, z"
coalesce(trim(nickname), name, 'invitado')

Uso por contexto

  • Condición: len(items) > 0
  • Asignación: x = toNumber('42')
  • Texto: Hola {{ upper(user_name) }}
  • JSON embebido: escapa comillas — {"key":"{{ value }}"}
Insertar ejemplo en el canvas

Crea un flujo mínimo con variables, conteo, join y toNumber:

  1. Start → define variable items como lista
  2. Assign: items = split('a,b,c', ',')
  3. Assign: items_count = len(items)
  4. Assign: csv = join(items, ';')
  5. Assign: n = toNumber('42')
  6. Response: Tienes {{ items_count }} elementos; csv={{ csv }}; n={{ n }}

Ayuda del Simulador (resumen)

📝 Variables

Interpolación en textos con {{variable}}:

Hola {{user_name}}, tienes {{user_age}} años
🎨 Markdown
  • **negrita**, *cursiva*, # Título, listas y `código`
📊 DataInfo

Permite construir JSON con variables interpoladas:

{"user":"{{user_name}}","age":{{user_age}}}
💡 Consejos
  • Las variables y Markdown se renderizan en el chat del simulador
  • Escapa comillas en JSON cuando sea necesario

Ejemplos por tipo de elemento

Start

Declara variables iniciales del flujo.

variables: [
  { "name": "items", "isList": true, "defaultValue": "[]" },
  { "name": "user_name", "defaultValue": "" }
]
Response

Muestra texto (array de strings). Soporta Markdown y {{ vars }}.

i18n: { es: { text: [
  "Hola {{ user_name }}", "Tienes {{ len(items) }} elementos"
]}}
Input

Pide un valor y lo guarda en una variable.

i18n: { es: { prompt: "Tu nombre" } },
save_as: "user_name"
Assign Variable

Asigna valores (expresiones soportadas: len, split, join, toNumber, bool...).

assignments: [
  { "target": "items", "value": "split('a,b,c', ',')" },
  { "target": "items_count", "value": "len(items)" },
  { "target": "csv", "value": "join(items, ';')" },
  { "target": "n", "value": "toNumber('42')" },
  { "target": "user.name", "value": "'Ana'" }
]
Condition

Evalúa una expresión booleana y redirige por true/false.

expr: "len(items) > 0",
true_target: { node_id: "res_ok" },
false_target: { node_id: "res_vacio" }
ForEach

Itera una lista y expone item y index.

type: "foreach",
source_list: "items",
item_var: "item",
index_var: "index",
loop_body: { node_id: "resp_item" }
Button/Choice

Ofrece opciones; cada opción apunta a un siguiente nodo.

options: [
  { "label": "Sí", "target": { "node_id": "n_ok" } },
  { "label": "No", "target": { "node_id": "n_ko" } }
]
REST Call

Realiza llamadas HTTP. Soporta {{variables}} en la URL.

method: "GET",
url: "https://api.example.com/user/{{ user_id }}",
save_as: "resp"
Set Goto

Fija la variable de salto (goto) para redirigir el flujo.

target: "nodo_destino"
Form

Recolecta varios campos en una sola interacción.

fields: [
  { "name": "email", "label": "Email", "type": "text" },
  { "name": "age", "label": "Edad", "type": "number" }
]
